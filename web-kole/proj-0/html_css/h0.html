Мәтінді форматтаудың мысалы

Бұл қарапайым қалың мәтін, <b> тегін пайдаланып.

Бұл қиғаш мәтін, <i> тегін пайдаланып.

Бұл асты сызылған мәтін, <u> тегін пайдаланып.

Бұл белгіленген мәтін, <mark> тегін пайдаланып.

Бұл кіші мәтін, <small> тегін пайдаланып.

Бұл маңызды мәтін, <em> тегін пайдаланып.

Аралас мысал: қалың, қиғаш, асты сызылған және маңызды мәтін бірге.
